# IT-Grundlagen-Quiz

## Beschreibung
Ein einfaches Konsolenquiz zu Themen der Fachinformatik Systemintegration.

## Start
python main.py

## Themen
- Netzwerk
- Hardware
- Betriebssysteme
- IT-Sicherheit
- Git und GitHub

## Zusammenarbeit
Jede Person hat auf einem eigenen Branch gearbeitet.
Änderungen wurden über Pull Requests erst in dev übertragen, dann in main übernommen.

## Probleme
Ein Branch wurde mit einer älteren Python-Version gepusht, dies konnte beim Review des Pull Requests gelöst werden.
